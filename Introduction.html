教會簡介

本教會成立於二零一七年七月廿五日正式開始拓展在香港新界北區天水圍探訪獨居長者及家庭, 拓展和長者培訓事工。探訪小組去關懷有需要的長者及家庭，已經決志信主多年，但沒有教會承擔牧養工作，作為神的僕人，我們經禱告及神的安排下，我們開始新界粉嶺打鼓嶺坪輋新村建立第一所教會，我們去服侍長者，精神病康復者及坪洋三鄉的居民，感謝讚美主，教會和信徒得到造就和支援乃主上帝所賜之褔和恩典。

本教會是主耶穌基督建立、信仰純正、沒有任何宗派背景，並堅守聖經信仰的社區教會、我們完全獨立、也沒有任何宗派支持、憑信心靠主維持。

教會信仰

本教會確信新舊約聖經為三位一體獨一真神的啟示，全部由聖靈感動而寫成, 真實可信, 並無錯誤。本教會認同各宗派教會的信仰，並接受使徒信經為信仰的準則。

教會的唯一根基是主耶穌基督(太16 : 16 ; 林前3 : 11)。教會要遵行「大使命」，使萬民作主門徒(太28:19-20)。教會是永生神的家，真理的柱石和根基，要傳講持守褔音真理(提後3:15);為要成全聖徒，各盡其職，建立基督的身體(弗4 : 11)，使之成為榮耀的教會，聖潔沒有瑕疵, 完全歸屬於主(弗5 : 25-27)。

教會是建造在使徒和先知的根基上，有主基督耶穌自己為房角石(弗2 : 20)。換言之，教會是建造在主耶穌基督的聖道啟示上，即透過使徒知寫成的聖經。神用真道重生了我們，也是以至聖真道來造就我們(雅1 : 18 ; 彼前1 : 23-25 ; 猶20)。所以教會的生活事奉，必須完全遵行聖經，不可高舉人的傳統、理性、經驗 ; 毫不摻雜人本主義，不容世俗作風滲入。

教會使命

我們靠著主耶穌基督的恩典在社區中設立教會，並建立關懷中心，去服待有需要的長者及家庭。因為我們接受主耶穌基督的教訓：在生命，人比任何東西都重要！因為上帝愛世人。

"神愛世人，甚至將他的獨生子賜給他們，叫一切信他的，不致滅亡，反得永生。" (約翰福音三章十六節)

本教會願景

恩祐（深水埗堂）基金會願景是成為一間具高度熱誠及重視價值的綜合服務團體，期望在全人服務、提升質素、服務創意、為服務對象解難、支援家庭及建設社區各方面達至平衡卓越表現, 並與海外或國內教會, 慈善機構及學校舉辦活動。促進學術, 文化, 環保, 科學創新及基督教福音交流, 讓區內人士擴闊思維, 增廣見聞。展望不久將來, 恩祐(深水埗堂) 基督教會能落實啟動, 成就傳福音的大使命。
